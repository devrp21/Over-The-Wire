### **Step 7: Create a Python Decryption Script**  
Create a `dec.py` file with the following content:

```python
key = 'EICTDGYIYZKTHNSIRFXYCPFUEOCKRN'
cipher = 'PNUKLYLWRQKGKBE'

pt = ''
for i in range(len(cipher)):
    tmp = ord(cipher[i]) - ord(key[i])
    if tmp < 0:
        tmp += 26
    tmp += ord('A')
    pt += chr(tmp)

print(pt)
```
